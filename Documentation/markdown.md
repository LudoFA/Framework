
Les bases du Markdown
=====================

Pour faire un `h1`
~~~
Mon titre
=========
~~~

Pour faire un `h2`
~~~
Mon titre
---------
~~~

Pour faire un lien 

~~~
 [Texte du lien](url,texte)
~~~

Pour faire une liste simple `<ul>`
* Element 1
* Element 2
* Element 3

~~~~~
* Element 1
* Element 2
* Element 3
~~~~
